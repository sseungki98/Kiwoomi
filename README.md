## 기본 제공 기능
- 주식 이름 혹은 코드 입력 시 관련 정보 출력 (현재가, 체결가, 실시간 종가, 전일 대비, 등락률, 시가, 종가 등)
- 조건 검색에 사용될 조건 입력 시 관련 정보 출력
- 주식과 관련된 뉴스나 정보 제공
- ❌ 자동매매에 관한 기능은 제공하지 않음

## How to use?
1. 텔레그램에서 @dku_kiwoomi_bot을 검색한다.
2. 원하는 명령어를 입력하여 관련 정보를 얻는다.(명령어는 문서 확인 혹은 /help를 통해 확인할 수 있음)
